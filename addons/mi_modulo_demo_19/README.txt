Mi Módulo Demo 19
=================

Este addon de ejemplo para Odoo 19 incluye:
- modelo: mi.modulo.demo.record
- vistas: list, form y search
- menú
- permisos
- datos demo
- botón Python

Instalación:
1. Copia la carpeta mi_modulo_demo_19 a tu ruta de addons personalizados.
2. Reinicia Odoo.
3. Activa el modo desarrollador.
4. Apps > Update Apps List.
5. Busca "Mi Módulo Demo 19" e instálalo.

Nota:
Si no aparece, revisa que el addons_path incluya tu carpeta personalizada.
